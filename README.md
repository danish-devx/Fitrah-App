A modern and clean **Fitrah Calculator Web App** built to calculate *Zakat ul Fitr* easily for Ramadan 1447.

This tool allows users to:

* Enter per person amount
* Add number of persons
* Automatically calculate total per commodity
* Instantly display grand total in PKR

* 🧮 Live Auto Calculation

* 🎨 Clean Neumorphism UI Design

## 📂 How It Works

1. User enters per person amount.
2. Adds number of persons.
3. App calculates row total.
4. All rows are added automatically.
5. Grand total is displayed in PKR format.

All calculations update instantly on input change.
